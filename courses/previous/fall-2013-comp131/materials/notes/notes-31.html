Figure out the profit or loss that would be experienced if the following trading strategy were used:

         Buy a specified number of shares the first time the price goes up.

         Hold them until the end of the period being analyzed and then sell them.

 

         Ex:  22 21 21 23 20 24 25 27                     Assume we buy 5 shares.

 

                  We need to find the price following the first price increase.

                  To find the increase:

                           Compare each price to the one before it.

                           Starting at index 1

                                    Compare 21 to 22 – no increase

                                    Compare 21 to 21 – no increase

                                    Compare 23 to 21 – there’s the first increase

                                    Make a note of the price:

 

buyPrice: 23

 

Now to compute the profit, subtract that from the final price and multiply by the number of shares being purchased.

        

profit: (27 – 23) * 5 = 4 * 5 = 20.

                  Label buy price, final price and shares bought.

 

Does that work if the increase is in the first 2 days?   Last 2 days?

 

What happens if there is no price increase?    27 26 24 20 18

Then the strategy would not buy any stock.

So the profit or loss should be 0.

We’ll ignore this case for now and address it later.

 

 

    /**

     * Compute the profit or loss that would be experienced if the following

     * trading strategy is used: Buy the specified number of shares on the

     * first price increase of the period.  Sell the shares at the end of the

     * period. If there are no price increases during the period then this

     * method should return 0.

     *

     * @param the number of shares to buy and sell.

     * @return the profit or loss experienced by using this trading strategy.

     */

    public double buyOnFirstIncrease(int shares) {

        // The price at which stock is bought

       double buyPrice = 0;

 

        // index of day for comparison to previous day

        int index = 1;

 

        // Check successive days until a price increase is found

        // or we run out of days.

  while (index < prices.length && buyPrice == 0) {

 

            // If an increase if found

if (prices[index] > prices[index-1]) {

              // Set the buyPrice

                buyPrice = prices[index];

            }

 

          index++;

       }

 

        // Compute and return the profit.

       double profit = (prices[prices.length-1] - buyPrice) * shares;

        return profit;

    }

 

When writing the code be sure to discuss the use of while vs. for.

         Typically we’ll use for when the number of iterations is known in advance.

                  E.g.    go through every element in the array.

                           go through the first 10 elements.

                           go through the last 1/2 of the elements.

 

         Typically we’ll use while when the number of iterations is not known in advance.

                  E.g.    go through the elements until an increase is found.

                           go through the elements until a specific value is found.

 

 

Now, how can we deal with the case where there is no increase in the price?

If there is no increase in the price then the while loop will exit without ever setting the buyPrice to something other than zero (because the condition

index < prices.length fails).

 

So we could add a check after the while loop to see if the buyPrice has been set.  If it has not been set then return 0, otherwise compute the profit or loss the normal way.

 

if (buyPrice == 0) {

    return 0;

}

else {

    return (prices[prices.length-1] - buyPrice) * shares;

}
