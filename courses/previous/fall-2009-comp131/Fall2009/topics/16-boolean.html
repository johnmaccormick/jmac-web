    • If the year is before 1384 and it is divisible by 4 it is a leap year.
    • Otherwise the year is a leap year if...

      • as long as it is not divisible by 100,

Homework Assignment 5c

  1. Consider the boolean expression: (x < 10) && (x > 4). What values of x make this expression false? Justify your answer.
  2. Consider the boolean expression: !(x < 10). What values of x make this expression true? Justify your answer.
  3. Consider the boolean expression: (year % 400 == 0) && (year % 100 != 0). What values of year make this expression true? Justify your answer.
  4. Download and unzip the GuessingGame project, and edit the class Game within that project. When finished, this class will allow two people to play everyone's favorite guessing game - "I'm thinking of a number betweem 1 and 10." One player will use the constructor for the class to create a new Game object containing the number they are thinking of. The second player will repeatedly call a method called guess to attempt to guess the number.

    You must implement this constructor and method as follows:

    • the constructor must take one parameter - the number that the first player is thinking of
    • the guess method must also take one parameter - the number that the second player is guessing. The method must then respond by returning one of the following messages:
      • "Your guess is out of range!" if the guess is not between 1 and 10
      • "You win!" if the guess is correct
      • "Your guess is too high!" if the guess is too high
      • "Your guess is too low!" if the guess is too low
    Stub versions of the constructor and guess method are provided for you, so you need only fill in the code in the bodies of these methods.

    You must also provide test cases for the guess method that achieve statement coverage.

    Submit this project via WebCAT. Note that WebCAT will not accept your submission if you have not created a test class for class Game.
