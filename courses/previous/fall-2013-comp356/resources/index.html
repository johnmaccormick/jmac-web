Instructions for editing and running Scheme programs:

  • To launch the right program:
    • On the Tome Macs: run Applications | PLT Scheme v4.2 | DrScheme.
    • On your own machine: download the latest version of DrRacket and run it.
  • Always set your language to "Advanced Student": go to Language | Choose language | How to Design Programs | Advanced Student.
  • To test that it works: use the File menu to create a new file, put some simple Scheme into it, such as "(+ 3 4)", save the file, then click Run. You should see the output in the Interaction Window.
  • [with thanks to Cooper Jackson for this useful tip] For debugging complicated list structures, it helps to turn off the display feature known as "sharing". To do this, select the "Advanced Student" language as described above, then click "Show Details", then uncheck the option for "Show sharing in values".

Class 26: Thursday, November 21

Instructions for using tracing facilities with XGP:

  • These instructions use the file course-inference.pl as an example. We assume you would like to trace the query teachingStudent(maccormick,gao).
  • Consult the file course-inference.pl, then evaluate the query "trace, teachingStudent(maccormick,gao)."
  • In the XGP Console window, repeatedly press the Enter key to step through the evaluation.
  • To terminate tracing, enter "n" (followed by Enter) in the XGP Console window.

The main topic for today is covered in section 16.7 of the textbook. Some brief additional notes on Prolog are provided. The accompanying file tree.pl demonstrates a non-trivial Prolog data structure.
