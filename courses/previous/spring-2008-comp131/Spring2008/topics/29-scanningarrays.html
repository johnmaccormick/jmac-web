Practice Exercises

  1. If it was not done as an example in class, study the getMaxPrice in the StockAnalyzer class. This method does not take any parameters and returns a double value indicating the maximum price for the stock during the period being analyzed.
  2. Add a method named getMinPrice to the StockAnalyzer class. This method does not take any parameters and returns a double value indicating the minimum price for the stock during the period being analyzed.
      Note: The StockAnalyzerTest class already contains test methods for each of the methods that you are writing for these exercises. You can run each test individually by right clicking on the StockAnalyzerTest class and choosing the appropriate test method.
  3. Add a method named getDayOfMaxPrice to the StockAnalyzer class. This method does not take any parameters and returns an int value indicating the index at which the maximum price occurs. If there are multiple days that tie for the maximum price the index of the most recent day with that price should be returned.
  4. Add a method named getDayOfMinPrice to the StockAnalyzer class. This method does not take any parameters and returns an int value indicating the index at which the minimum price occurs. If there are multiple days that tie for the minimum price the index of the most recent day with that price should be returned.
  5. Add a method named getLargestPriceDifference to the StockAnalyzer class. This method does not take any parameters and returns a double value indicating the largest difference between any two prices that occurred within the period being analyzed. Note that if the lowest price occurred before the highest price then the value returned by this method will be positive. If the highest price occurs first then the value returned by this method will be negative. Hint: use this to call other methods of the StockAnalyzer class to do some of the work.
  6. Add a method named isVolatilePeriod to the StockAnalyzer class. This method accepts one parameter, an int value specifying a number of days. This method then returns true if the maximum and minimum prices are separated by no more than the specified number of days of each other, and false otherwise.

  1. Complete the following exercises using the GradeBook project that you have been working with for the past two homework assignments. Note that if you had problems with the previous homework assignment, you should use the available solutions to correct your work before completing this assignment.
    1. Add a method named findHomeworkAverage to the GradeBook class. This method does not take any parameters and returns an int value indicating the average of all of the homework scores. Any fractional part of the average is truncated. Also add test methods that achieve statement coverage for your findHomeworkAverage method.
    2. Add a method named findHomeworkMin to the GradeBook class. This method does not take any parameters and returns an int value indicating the lowest homework score. Also add test methods that achieve statement coverage for your findHomeworkMin method.
    3. Add a method named findIndexOfHomeworkMax to the GradeBook class. This method does not take any parameters and returns an int value indicating the index of the highest homework score. If there are ties for the highest homework score, this method returns the index of the most recent homework with that score. Assume that older homework scores are stored at lower indices. Also add test methods that achieve statement coverage for your findIndexOfHomeworkMax method.
    4. Add a method named isHomeworkConsistent to the GradeBook class. This method takes one parameter that is an int value specifying the maximum allowable difference of the homework scores from the average homework score. This method then returns true if all of the homework scores differ from the average by no more than the specified maximum allowable difference, and false otherwise. Also add test methods that achieve statement coverage for your isHomeworkConsistent method.
