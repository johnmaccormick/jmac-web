    1. Add a method named getFirstMintCoin. This method does not does not accept any parameters. When invoked it returns the first Coin in the collection that is in mint condition. Note: For full credit you must use a while loop to complete this method.
    2. Create test methods for your getFirstMintCoin following the Testing Rules.

In other words any coin minted before the specified year is considered old. When invoked the method returns the last Coin in the collection that is considered old. If there are no Coins that were minted before the specified year, this method returns null. Be sure to include a javadoc comment describing your method. Hint: Use a while loop and start at the end of the list and work backwards until you find an old coin.
