  3. The following definition of class Widget contains at least three errors that would prevent the class from compiling. Identify each error, and explain why it is an error. Do NOT use BlueJ to find the errors. 
    public class Widget {
    private int stockNum;  // stock number used for tracking product
    private String name    // name of the widget
    int private quantity;  // number in the warehouse

    /** create a new Widget with the specific field values
      * @param initStockNum the stock number
      * @param initName the widget name
      * @param initQuantity the number in the warehouse
      */
    public Widget(int initStockNum, String initName, int initQuantity) {
        stockNum = initStockNum;
        name = initName;
        quantity = initQuantity;
    }

    /** set the stock number of the widget */
    public void setStockNum() {
        stockNum = initStockNum;
    }

    /** decrease the quantity by 1 */
    public void decreaseQuantity() {
        quantity = quantity - 1;
    }
}
