Practice Exercises

  1. Implement a Pet class that represents pets as follows:
    1. use BlueJ to create a new project called MyPet that is stored in your CS131 directory.
    2. create a new class called Pet in your MyPet project.
    3. in the Pet class, declare fields so that each instance of the class can keep track of the name, age (in months), weight, color and species of a pet. Make sure that your class compiles correctly. Note that the fields should all be private.
    4. add a constructor that initializes all fields of a Pet object using parameters. Compile your class, and create an instance of the class using the constructor that you just wrote.
    5. add a method called print that prints all information about a Pet on the screen. Create an instance of the class and use it to call this method.
    6. add a mutator method called increaseAge to increase the age of a Pet. This method should take the number of months to increase the Pet's age by as a parameter.
    7. add a method called getAgeInYears that returns the number of years old a Pet is, ignoring any fractional part. For example, if a Pet object having an age of 20 months is used to call this method, it should return 1.
    8. add a method called printAge that prints the age of the Pet in years and months. For example, if a Pet object having an age of 20 months is used to call this method, it should print 1 year and 8 months.
    Be sure to write javadoc comments for each constructor and method in your Pet class.

Homework Assignments

  1. Using the Account-v2 sample project, modify the print method so that it does not use + to concatenate any Strings, but still produces exactly the same output. Copy and paste your revised method definition into your answer for this exercise.

    Hints: use System.out.print, rather than System.out.println. The following code: 

    System.out.print("Using print");
System.out.println("instead");
System.out.print("of println");
    will produce this output: 
    Using print instead
of println.

    You can also print out the value of a field by passing it as a parameter to print. For example: System.out.print(balance); will display the value of the balance field.
