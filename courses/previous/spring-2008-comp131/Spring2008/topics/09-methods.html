  2. The following definition of class Widget contains at least 4 errors that would either prevent the class from compiling, or cause the class to work incorrectly when run. Identify each error, and explain why it is an error. Do NOT use BlueJ to find the errors. 
    public class Widget {
    private int stockNum;  // stock number used for tracking product
    private String name;   // name of the widget
    int private quantity;  // number in the warehouse

    /** create a new Widget with the specific field values
      * @param initStockNum the stock number
      * @param initName the widget name
      * @param initQuantity the number in the warehouse
      */
    public Widget(String initStockNum, String initName, int initQuantity) {
        stockNum = initStockNum;
        initName = name;
        quantity = initQuantity;
    }

    /** set the number of the widget in the warehouse */
    public void setQuantity(int newQuantity) {
        quantity = newQuantity;
    }

    /** get the stock number
      * @return the stock number */
    public String getStockNumber() {
        return stockNum;
    }

    /** decrease the quantity by 1 */
    public int decreaseQuantity() {
        quantity = quantity - 1;
    }
}
