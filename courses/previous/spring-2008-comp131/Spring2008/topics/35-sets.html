Practice Exercises

Use the StudentDatabase-v2 project to complete the following refactorings:

  • update the toString and print methods of the Student class to work correctly with the changes from the practice exercises of the last class meeting.
  • add a method called deleteCourseByInstructor to the Student class. This method should take the name of an instructor as a parameter, and should delete the first course taught by that instructor from a student's schedule. Note that the HashMap class does have a remove method - consult the javadoc documentation for HashMap to see how to use this method.
  • record test cases for deleteCourseByInstructor method. Note that you can check whether a course is in a student's schedule by using the isTaking method.

Homework Assignments

Using the LibraryDB project that you have been developing for the last three homeworks, complete the refactoring that you began in the previous homework. That is, update the hasBookByAuthor, deleteBookByAuthor, print and countBooksBySubject methods of the Library class to work correctly with the books field as a HashMap.

Hint: comment out the method you had previously added when refactoring hasBookByAuthor and deleteBookByAuthor (for finding the position of a Book by author), and replace it with a method that takes an author's name as a parameter and returns a call number or an indication that the Book was not found. Be sure to include a javadoc comment for this new method.

NOTE: for any method definition that you need to change, do NOT delete the previous version of the method body. Instead, simply comment it out using /* and */ comments so that it will be ignored by the compiler, and then put the new version of the method below the old version. Failure to comply with these instructions will result in an immediate 15 point deduction.

Hints:

  • the HashMap class does have a remove method - consult the documentation to see how the method works.
  • run your tests to verify that the program still works correctly.
