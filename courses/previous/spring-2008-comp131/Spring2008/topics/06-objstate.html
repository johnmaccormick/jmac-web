Homework Assignments

Continue using the Car project to answer the following questions.

  1. Which field of a Car object changes when you call the setMake method?
  2. What is the type of the color field? Of the year field?
  3. Do the types of the color and year fields make sense for the aspects of real world cars that they represent? Why or why not?
  4. Modify the source code of the Car class so that when a new Car object is created, it represents your favorite real world car. Create a new Car object and verify that it now represents your favorite car. Include all of the lines of source code that you changed as your answer to this problem. Note that you can copy and paste code from the BlueJ editor to other applications.
