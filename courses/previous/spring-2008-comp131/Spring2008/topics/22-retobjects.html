Practice Exercises

  1. Fill in the body of the moveDVDToEnd method in the VideoStore class.
  2. Create test methods for your moveDVDToEnd method. This method both removes from and adds to a collection. Thus your test methods should follow the Testing Rules for both situations.
  3. Fill in the body of the getDVDIfInStock method in the VideoStore class.
  4. Create test methods for your getDVDIfInStock method. Be sure that your test methods achieve statement coverage for your method.

Homework Assignments

  1. Perform the following tasks using the CoinCollection project that you began working with in the last homework.
    1. Add a getCoin method to the CoinCollection class. This method must accept an integer parameter indicating the position of the Coin to get. If the position specified by the parameter is valid, this method returns the requested Coin. If the position is not valid (i.e. it would cause an IndexOutOfBoundsException, this method prints an error message and returns null. You must also include a javadoc comment above this method describing what it does. Use the comments above the methods in the VideoStore class as a model.
    2. Create test methods for all of the methods in your CoinCollection class. You will find it useful to create a test fixture before proceeding with these tests. It is suggested that you create a test fixture containing 3 or 4 Coin objects and a CoinCollection object. In completing your tests you should use the appropriate Testing Rules from class.

    Save this project. You will be adding additional methods to it in the next several homework assignments. Once all of the assignments are complete, you will submit your project via Web-CAT.
