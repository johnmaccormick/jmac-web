Thursday, November 12

Required reading: Chapter XVII of GEB ("Church, Turing, Tarski and Others"). Summarization homework is not required. Instead, come to class prepared to answer the following question: for each of the 10 versions of the Church-Turing thesis given in this chapter, do you believe the given version is true or false?

The following definitions will help you understand the reading:

  • isomorphism: a fixed way of translating between one formal system and another (recall the simple example given in class, of an isomorphism between arithmetic written with two different ways of writing the digit 7)
  • FlooP: a simple computer programming language used for some of the examples in GEB. You don't need to know anything about this language, except that it can be used to program computers.
  • TNT: this stands for "typographical number theory". It is the particular formal system that is used in GEB to describe arithmetic.
