Extra Problems: Conditionals

  1. Open BlueJ and, unless you have one already, create a new project called ExtraProblems. Then create a new class within your project called Conditionals. This class does not need to have any fields or a constructor to initialize them.

  2. A nonnegative integer K is divisible by a positive integer L, if the remainder after dividing K by L is zero.

    Write a method called divisible() in your Conditionals class that accepts two arguments, M and N. It should then return a boolean value reporting whether either of the input values is a divisor of the other. Note that when you call this method by selecting it from the BlueJ interface, a window will automatically pop up in which you can type the argument values.

  3. Add the following method to your Conditionals class. Then experiment with it and explain why the code probably does not do what the programmer intended. 

      public String mystery(int num) {
     //if even, increment; if odd, decrement
     if (num % 2 == 0) {
        num++;
     }
     if (num % 2 != 0) {
        num--;
     }

     String result = "Result = " + num;
     return result;
  }

  5. Classifying Triangles:
    [This problem is based on an exercise by Darrah Chavey, Beloit College.]

    Write a method that accepts the lengths of three sides of a triangle as parameters, and then returns the classification of the triangle as follows:

    • invalid input: all sides must have lengths that are strictly positive
    • not a triangle: the sum of any two sides must be greater than the length of the third
    • scalene triangle: no two sides have equal length
    • isosceles triangle: two sides (but not the third) are equal
    • equilateral triangle: all three sides are equal
