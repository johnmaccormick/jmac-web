COMP 131 - Macintosh Tips

  • Menubars: On Macintosh computers, the application menu bars appear at the extreme top of your monitor. They do not appear at the top of the application window, where you may be used to them from Windows or Linux applications.

  • Keyboard "hot keys": Many of these key combinations are as you would expect on Windows, except that you use the command key instead of control. For example:
    • cut: command-x
    • copy: command-c
    • paste: command-v
    • save: command-s

  • Mouse right-click: Macintosh mice look like they have only one button, but there are really two inside. By default, both buttons do the same thing (left-click), but you can configure your account to allow you to right-click as well. To do so:
    • click on the apple in the upper-left corner of your screen
    • select System Preferences, and then Mouse (in the 2nd row of icons)
    • in the dropdown box pointing to the right mouse position, select "Secondary Button"
    • close the Mouse window

    Alternatively, you can simulate a right-click by pressing the control key while clicking the (one) mouse button.


  • To view your files: Use the Finder application. Its icon looks somewhat like a square blue face, and it is located in the tool bar at the bottom of your screen. You should see a folder labeled "private" in the left panel of the Finder -- this is your Y: drive.

  • To force a "frozen" application to quit: Press the option, command, and esc keys simultaneously.

  • To log out: Click on the apple icon in the upper-left corner of the screen, then select Log Out.


  • Calculator: There is a calculator application in the Applications folder. To make the square-root button visible, first launch the calculator application, and then select Scientific from the View menu.


  • To create a zip file (for example, to zip a BlueJ project):Use the Finder application to locate the folder your BlueJ project is stored in. Right-click on the folder, and select Compress "folderName." This will create a new file called folderName.zip.
