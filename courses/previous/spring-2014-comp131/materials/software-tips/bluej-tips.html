COMP 131 - BlueJ Tips

  • To print source code: To print the source code of a class, open the source code editor for that class and then select Print from the Class menu on the main menubar.

  • To enable unit testing tools:
    • Macintosh computers: Select BlueJ > Preferences from the menubar. Then click on the Miscellaneous tab at the top of the dialog window that appears. Then check the box labeled "Show unit testing tools" near the center of the window, and click OK.

    • Other computers: The steps may be slightly different on other computers. For example, on a Linux machine you will find Preferences under the Tools menu instead of the BlueJ menu.

  • To create a test class: Right-click on the class you want to test. Then select "Create a Test Class."
