Homework Assignment 7d

  1. Make the following additions to the CoinCollection project that you have been working with for the last several assignments. For each method, be sure that you use the specified name and that your method signatures are consistent with what is described below.
    1. Add a method named getTotalValue. This method returns a double value indicating the total value of all of the coins in the collection. It does not accept any parameters. Be sure to include a javadoc comment describing your method.
    2. Create a test method for your getTotalValue method.
    3. Add a method named countCoinsOlderThan that returns the number of Coins that were minted during or before a specified year. This method takes one parameter indicating the year. If the CoinCollection does not contain any Coins minted during or before the specified year then this method should return 0. Be sure to include a javadoc comment for your method.
    4. Develop test methods for your countCoinsOlderThan method. Your tests must achieve statement coverage.
    5. Add a method named getAverageAge that that takes one parameter indicating the current year and returns an int indicating the average age in whole years of the Coins in the collection. The age of a Coin is the current year minus the year that it was minted. If the CoinCollection does not contain any Coins then this method should return -1. Be sure to include a javadoc comment for your method.
    6. Develop test methods for your getAverageAge method. Your tests must achieve statement coverage.
