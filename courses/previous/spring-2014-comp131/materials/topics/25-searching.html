Homework Assignment 8a

  1. Using the CoinCollection project that you have been working with for the last several homework assignments, perform the tasks described below. NOTE: If you had difficulties with the previous homework assignment be sure to use the solutions posted outside the instructor's office to correct that assignment before beginning this one.
    1. Add a method named increaseCoinValues. This method does not return a value. It accepts one parameter of type double indicating the percentage by which the value of each coin should be increased. Note that, even though the percentage parameter is a double, you should expect values that are greater than one. For example, if the percent increase is to be 12.5% the argument that is passed should be 12.5 (not 0.125). When invoked the method increases the value of each coin by the specified percentage. Be sure to include a javadoc comment describing your method.
    2. Create a test method for your increaseCoinValues method. Be sure to follow the Testing Rule for a method that modifies the elements of a collection.
    3. Add a method named getMostValuableCoin that returns the Coin with the highest value. If there is a tie for the most valuable Coin, then this method returns the first Coin with that value. If the CoinCollection does not contain any Coins then this method should return null. Be sure to include a javadoc comment for your method.
    4. Develop test methods for your getMostValuableCoin method. Be sure to follow the Testing Rule for a method that searches the elements of a collection.
    Save this project. You will be adding to it in the next two homework assignments. Once those assignments are complete, you will submit your project via Web-CAT.
