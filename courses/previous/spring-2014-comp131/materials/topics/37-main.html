Troubleshooting

Can't run java or javac from the command line? Are you getting an error like "not recognized as an internal or external command"? If so, fix your environment variables by running the following two commands (copy, then paste them into your command prompt window using a right-click):

setx PATH "%PATH%;C:\Program Files\Java\jdk1.6.0_21\bin"
setx CLASSPATH "C:\Program Files\Java\jdk1.6.0_21\lib;."

Notes: You might need to slightly alter the above, depending on where Java is installed on your machine. You may need to launch a new command prompt window to see the effects. Seek help from your instructor if necessary. Also, these instructions are for Windows machines only.
