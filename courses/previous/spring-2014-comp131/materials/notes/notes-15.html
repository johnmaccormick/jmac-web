Recap: Conditional (a.k.a. if) Statements:        

These statements allowed our programs to check a condition and if that condition was true do one thing, otherwise if the condition was false do something else.

 

Statement Coverage:

          What is statement coverage?

It is our objective in developing test cases. We want to ensure that the set of tests that we develop executes every statement in our program at least once. In other words, the test cases cover the statements of the program.

Testing max3:

         “Code-a-little, Test-a-little”

 

         How many test cases will we need to achieve statement coverage for the max3 method?

Four.  Each time max3 is called; exactly 1 of the 4 assignment statements will be executed.  We need to ensure that every one of them executes at least once.  So we need at least 4 test cases.

 

Test Plan:

         Case 1:       executes max = num1.

                  What values can I use?

                           num1 > num2   and   num1 > num3

 

setup:       FindMax(5, 4, 3)

Cascading if Example:

 

         Overview of GradeCalculator class:

                  Computes the letter grade for an assignment.

                  1 field, the number of points available on the assignment.

                           1 constructor initializes the number of points on the assignment.

grade method – returns the letter grade corresponding to a specific score on the assignment.

/**

 * return the letter grade associated

 * with a particular score

 */

public String grade(int score) {

int pct = score * 100 / numPoints;

// percentage on the assignment

 

String letGrade; // the letter grade on the assignment

 

if (pct >= 90) {

letGrade = "A";

} else if (pct >= 80) {

letGrade = "B";

} else if (pct >= 70) {

letGrade = "C";

} else {

letGrade = "other";

}

 

return letGrade;

}

Explanation:

                  First statement computes the score on the assignment as a percentage.

                  Define a local variable that will hold the letter grade.

The if statement (which I’ll come back to) sets letGrade to the correct letter grade based on the percentage.

                  Then return the letter grade.

 

                  This if statement is a cascading if statement.

In a cascading if, the computer checks the first condition, if it is true it executes the body of that if, then skips everything else.  If that condition is false, it skips the body and checks the condition of the next if.  If that condition is true, it executes the body and skips everything else.  And so on.
