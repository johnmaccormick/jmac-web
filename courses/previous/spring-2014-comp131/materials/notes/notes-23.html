Solution is a Looping Statement

Looping statements allow us to repeat a block of program instructions over and over again some number of times or until a termination condition applies.

 

For example, repeat the process of getting and printing out a DVD once for each DVD in the collection.

 

         Java has 3 looping statements:

         for loop           (today)

while loop               (soon)

                  for each loop   (maybe later)  Textbook uses this, but we will delay it.

 

The for Loop:

 

    for (<initializer>; <loop condition>; <update>) {

       <loop body>

}

 

 

         <initializer>: Executed before first iteration of the loop.

                  Like:    int day = 0;

 

<loop condition>: A boolean value - checked before each iteration of the loop.

                  Like:     day < prices.length

 

         <update>: Executed after each iteration of the loop.

                  Like:          day++;  (same as  day=day+1;)

 

Just like with the while loop, the loop body is executed over and over again as long as the <loop condition> is true.

 

Here’s little example.

 

    for (int num = 0; num < 5; num=num+1) {

       System.out.println(2*num - 1);

}

         What about if the update were changed to num=num+2?

         What about if the condition were num > 5?

         What about if the update were num=num-1?
