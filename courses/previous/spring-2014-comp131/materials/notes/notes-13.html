Let’s now look at the code of the rotate method.

        

Local Variable:

         Variable declared inside a method body.

         Scope is limited to the method body.

         Declared as: <type> <name>      (e.g. int newAngle;)

         Used for storing intermediate results (e.g. newAngle here.)

 

         Think of it like a register with whatever name you give it.

It is a place to store intermediate results and then refer to those values by name.

 

         We can also combine the declaration of a local variable with its initialization:

        

                  int newAngle = angle + degrees;

 

Now “Code-a-little, Test-a-little!”:
