Topic #27 – Returning a Collection

 

The search methods created last class found the DVD meeting a particular criterion:

e.g.    the first overdue DVD

the DVD with the most rental nights

the most overdue DVD

 

Sometimes it makes more sense for a search to return a whole collection of things:

         e.g.    Find all of the DVDs that are overdue.

                  Find all R rated DVDs

                  Find all DVDs that are more than 3 days overdue.

 

Let’s develop a method for finding all DVDs with a specified rating:

 

public VideoStore getAllDVDsWithRating(String rating) {  }

 

javadoc comment.

 

/**

* Find all of the DVDs with the specified rating and

* return them in a new VideoStore.   The name of the returned

* VideoStore will be "? Rated DVDs" where ? is the specified

* rating. If there are no DVDs with the specified rating then an

* empty VideoStore is returned.

*/

 

The plan for implementing this method:

         Create a new empty VideoStore

         Iterate through each of the DVDs in the original collection

         If a DVD has the specified rating then add it to the new VideoStore.

         Return the new VideoStore.          

 

public VideoStore getAllDVDsWithRating(String rating) {

     // Create a new empty VideoStore

    

     // Iterate through the DVDs

         // If the DVD has the specified rating

         // add it to the new collection.

 

     // Return the new collection.

}

 

To create a new VideoStore we need to invoke the constructor for the VideoStore class:

 

// Create a new empty VideoStore

VideoStore withRating = new VideoStore(

          rating + “ Rated DVDs”  );

 

When we invoke the constructor in code we must provide a value for each parameter, just as when we do it by hand. 

 

public VideoStore getAllDVDsWithRating(String rating) {

// Create a new empty VideoStore

VideoStore withRating = new VideoStore(rating +

“ Rated DVDs”);

       

// Iterate through the DVDs

for (DVD aDVD : dvdList) {

                      

         // If the DVD has the specified rating

// add it to the new collection.

String curRating = aDVD.getRating();

if (curRating == rating) {

withRating.addDVD(aDVD);

}

else {

     // do nothing.

}

}

       

// Return the new collection.

return withRating;

}

 

Can’t use  == to compare Strings!!!

Needs to be written as       curRating.equals(rating)
