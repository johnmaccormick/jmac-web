Topic #5:  Objects and Classes

 

Today we begin our study of programming using the Java programming language

 

Java is a high level language

Which means that it will be closer to English than machine or assembly language.

However, everything that we do in Java is ultimately being carried out by the computer manipulating 1’s and 0’s in much the same way as the knob & switch computer that we studied.

 

It was invented about 12 years ago by researchers at Sun Microsystems.

It is now used extensively in industry and on-line.

 

Java is also what is known as an Object Oriented (OO) programming language

 

In an OO language programs are constructed by using Objects

 

Objects in a program represent “things” in the real world:

         E.g.   Software for a Bank might have objects that represent:

Savings Accounts, Customers, Checking Accounts

                  Software for an auto auction might have objects that represent:

                  Cars

         Software for making drawings might have objects that represent:

                  Circles, Rectangles, Lines

                  etc.... any of the shapes that appear on the screen.

An operating system such as Windows or Mac OS have objects that represent:

                  Windows, Buttons, Menus

                  Etc…

 

In an OO language every object belong to something called a Class

         All objects of the same type belong to the same class.

         E.g.  

                  In a Bank System:

An object representing “My Bank Account” and an object representing “Your Bank Account” would both be objects that belong to the class Bank Account.

 

                  In an Auto Auction System:

An object representing a “Yellow 1976 Dodge Dart” and an object representing a “Red 2005 Ford Mustang” would both be objects that belong to the class Car.

 

We also will often use the word instance to talk about objects that belong to a class.

An object representing a “Yellow 1976 Dodge Dart” is an instance of the Car class.

 

So a Class defines a whole category of objects, while an object is a specific instance of that class.

         In the following statements which refer to an object and which to a class?

 

                  I have $44 in my CheckingAccount.

                  Most bank customers have a CheckingAccount.

                 

                  The library contains a lot of Books.

                  There is a copy of Moby Dick in the library.

 

                  Circles are round and can be filled with color.

                  The drawing on my screen contains a Circle.

         What is a method?

                  A method is an action that can be performed on an object.

                  Examples?

        

                  We’ll  often talk about “calling” or “invoking” methods.

                  e.g. You invoked the makeVisible method to display the object.

 

         What are parameters?

Parameters are values provided when a method is called that affect what the method does.    

 

 

How do you know when a method needs parameters?

The method signature indicates the parameters between the parentheses following the method name.

        

                  Each parameter listed has 2 parts, what are they?

The first part is the data type.

The second part is the parameter name.

 

         What is the purpose of the data type?

The data type defines what type of data is allowed to be used for the parameter.
