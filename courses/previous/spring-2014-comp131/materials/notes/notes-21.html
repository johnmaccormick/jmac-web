Java provides a number of Collection Classes that are useful for storing and manipulating collections of objects.

 

The ArrayList class is one such Collection Class:

 

ArrayList has one constructor and 4 methods that we are going to be concerned with today:

         ArrayList<type>()

                  Construct an empty ArrayList that can hold objects of the specified type.

add(<object reference>);

                  Add the object to the end of the collection.

     size();

                  Find out how many objects are in the collection.

get(<index>);

                  Get the object at the specified index (0 <= index <  size)

                  index º location

remove(<index>);

                  Remove the object at the specified index from the collection.

 

We are going to use the power of abstraction to allow us to use the ArrayList class and each of these methods.

A little more about new:

 

new: Used to create a new object and invoke its constructor.

 

         e.g.:

         theList = new ArrayList<DVD>();

         myAccount = new Account(1234, 5);

                           Values in the parentheses are passed to the constructor.

                 

The import statement:

This statement is needed because the ArrayList class, while provided with Java, is not automatically available for every class to use.

 

ArrayList is part of the Java Class Library

The Java class library contains 100s or 1000s of classes that have been written and tested and are ready to be used by your programs.

 

         The classes in the Java Class Library are organized into Packages

                  Each package contains a number of related classes.

                  ArrayList is in the java.util package

 

import java.util.ArrayList

Means make the class ArrayList from the package java.util available for use in this class.

 

This import statement must be included in any file that uses the ArrayList class.
