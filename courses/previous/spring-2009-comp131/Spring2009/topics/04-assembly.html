The Knob & Switch Computer with Assembly Programming
The final version of the Knob & Switch Computer can be programmed using assembly language instructions which are then translated down to their equivalent machine language instructions. For example, the instruction for loading the contents of Main Memory cell 13 into Register R2 is "LOAD R2 13" as opposed to the cryptic machine language instruction "1000000000101101" The assembly language instructions are stored in Main Memory along with data, making this a complete stored-program computer. Thus, you will have to enter instructions into memory locations just as you would data values.
