Practice Exercises

  1. Using the StockAnalyzer project complete the following exercises:
    1. Complete the maxProfitPerShare method in the StockAnalyzer class. This method does not accept any parameters and returns a double value indicating that maximum profit that could be made using a single buy and a single sell of the stock being analyzed during the period being analyzed. If it is not possible to make a profit on the stock during the period being analyzed, this method returns 0. Complete this method without using this to call any methods of the StockAnalyzer class. Note: The StockAnalyzer project contains test methods that you can run to check if your solution to this exercise works correctly.
    2. Complete the method named sillyTrader in the StockAnalyzer class. This method accepts a single parameter indicating a number of shares of stock. It returns the profit or loss that would be experienced if the following trading strategy is used: Buy the specified number of shares on the first day of the period, sell the specified number of shares on the second day of the period, buy again on the third day, sell again on the fourth day, etc... You can assume that there are at least two days in the period and that the number of days in the period is even. Note: The StockAnalyzer project contains test methods that you can run to check if your solution to this exercise works correctly.

Homework Assignment 9c

  1. Continue using the GradeBook project from the previous homework assignment to complete the following exercises.
    1. Add a method named hasDuplicateScore to the GradeBook class. This method checks to see if the student received the same score on a homework and a quiz. If any homework score is equal to any quiz score this method returns true, otherwise it returns false.
    2. Create test methods for the hasDuplicateScore method that you added to the GradeBook class. Your test methods must achieve statement coverage.
    3. Add a method named findBiggestHWJump to the GradeBook class. This method does not take any parameters and returns an int value indicating the biggest positive difference in scores between any two successive homework grades. For example if the homework scores are {75, 80, 97, 78, 68} then the result of this method would be 17, because the jump from 80 to 97 is the largest jump. If the homework scores contain no increases (e.g. {95, 87, 87, 53}) then this method should return -1.
    4. Create test methods for the findBiggestHWJump method that you added to the GradeBook class. Your test methods must achieve statement coverage.
