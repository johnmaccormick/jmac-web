Practice Exercises

  1. Add accessor methods for the checking and savings account balances of the customer to the BankCustomer class. These methods should simply return the balance of the appropriate account.
  2. Create test methods for the accessor methods that you added in the previous exercise. Use a test fixture for your test cases.
  3. Add a method to the BankCustomer class to transfer money from the customer's checking account to the customer's savings account. Your method must print an error message if sufficient funds are not available for the transfer, and not change any account balances in this case.
  4. Write test cases that achieve statement coverage for the transfer method that you wrote in the previous exercise. Use the test fixture that you created earlier for your test cases.

Homework Assignment 6c

  1. Using the Bicycle project that you started working on in the previous homework assignment:
    1. complete the definitions of the remaining methods in the class
    2. write test cases that achieve statement coverage for the setFrontTireSize, setRearTireSize, getSeatSize and getSeatColor methods
    3. submit your Bicycle project via WebCAT
  2. Modify the withdrawChecking method of the BankCustomer so that it has overdraft protection as specified in the following:
    • if the checking account has sufficient funds for the withdrawal, then the checking balance should be updated as usual
    • if the checking account does not have sufficient funds for the withdrawal, but the combination of the checking and savings accounts is sufficient, then all funds should be withdrawn from the checking account and the remainder of the funds needed to cover the withdrawal should be withdrawn from the savings account
    • if the combination of the checking and savings accounts does not contain sufficient funds for the withdrawal, then an error message should be printed and no balances should be modified
    Copy and paste your modified withdrawChecking method as your answer to this exercise.
  3. Write test cases that achieve statement coverage for the method you defined in the previous exercise. Your test cases MUST use a test fixture containing at least 3 objects. Copy and paste the test methods and the setUp method from the test class as your answer for this exercise. (The setUp method contains the code used to create the test fixture.)
