The Binary System
As the authors themselves put it "A pretty damn clear guide to a quite confusing concept by Christine R. Wright with some help from Samuel A. Rebelsky." The section on 'Basic Concepts Behind the Binary System and 'Conversion from Decimal to Binary' is most relevant to what we did today in class.

The Knob & Switch Computer with Control Unit
The this version of the Knob & Switch Computer adds a Control Unit allows for up to 5 instructions to be entered using the 1's and 0's of machine language. The patterns of 1's and 0's in the machine language instructions correspond to the knob and switch settings in the CPU . You can experiment with this extended simulator by entering machine language instructions directly, or it is also possible generate the instructions automatically by setting the knobs and switches then clicking on the green arrow next to an instruction. You can play with this version of the Knob & and Switch simulator until you have a better feel for how machine language instructions can control the operation of the computer's CPU and memory.

Practice Exercises

  1. For binary numbers:
    1. How are each of the following decimal numbers represented in binary? 2 12 16
    2. What decimal values are represented by the following binary numbers? 101 1000111
  2. For the Knob & Switch Computer with Control Unit:
    1. What machine language instruction specifies moving the contents of Main Memory cell 7 into Register R0?
    2. Give the machine language instruction that subtracts the contents of Register R1 from the contents of Register R0 and stores the result in Main Memory Cell 4. In pseudocode: MM[4] = R0 - R1

Homework Assignment 1c

  1. For binary numbers:
    1. How are each of the following decimal numbers represented in binary? 5 17
    2. What decimal values are represented by the following binary numbers? 100001 1101111
    3. Using 1 bit it is possible to write down 2 different patterns of 1's and 0's (e.g. 0 and 1). Using 2 bits, how many different patterns of 1's and 0's can you write down? What are they? Using 3 bits what are the patterns you can write down? Hint: each of the patterns using 2 bits can have a third bit added to them and the third bit can be either a 1 or a 0.
  2. For the Knob & Switch Computer with Control Unit:
    1. List a sequence of machine language instructions (a machine language program) for adding the contents of Main Memory cells 1 and 2 and storing the result in Main memory cell 3. In pseudocode: MM[3] = MM[1] + MM[2]
    2. Write a machine language program that stores 4 times the number in Main Memory cell 1 into Main Memory cell 2. (Hint: it is easy to double the value of a number stored in a register.)
