  2. Using the CoinCollection project, perform the tasks described below.
    1. Add a method named getCoinsInCondition to the CoinCollection class. This method should return a CoinCollection containing all coins in a specified condition (e.g. "good", "fair"). If there are no coins in the specified condition, this method should return an empty CoinCollection. The name of the returned CoinCollection will be the condition of the Coins that it contains (e.g. "good"). Be sure to include a javadoc comment for your method.
    2. Create test methods for your getCoinsInCondition method. These test methods should achieve statement coverage and should also follow the Testing Rule for methods that return a collection.
    3. Add a method named getCoinsMintedBefore to the CoinCollection class. This method should return a CoinCollection containing all Coins minted before a specified year. If there are no Coins minted before the specified year, this method should return an empty CoinCollection. The name of the CoinCollection that is returned will be "Minted Before <year>" where <year> is replaced by the specified year. Be sure to include a javadoc comment for your method.
    4. Create test methods for your getCoinsMintedBefore method. These test methods should achieve statement coverage and should also follow the Testing Rule for methods that return a collection.
