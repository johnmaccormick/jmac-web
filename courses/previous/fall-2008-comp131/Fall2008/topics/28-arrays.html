Homework Assignment 9a

Download and unpack the GradeBook project and use it to complete the following exercises. Javadoc comments and method signatures have not been provided for the methods that you will be adding to this class. As you add methods, please be sure that names of the methods that you write exactly match those given in the assignment. In addition, if a method has more than one formal parameter, be sure they are listed in the order described in the assignment. Following these rules will ensure that the reference tests used by WebCAT will compile with your implementation of the GradeBook class. Finally, you need to provide a javadoc comment for each of the methods that you write.

  1. Define the fields for the GradeBook class.
  2. Implement the constructor for the GradeBook class.
  3. Implement an accessor named getStudentName that returns the name of the student.
  4. Implement accessor methods named getNumHomeworks and getNumQuizzes. These methods return integer values indicating the total number of homework scores or quiz scores that the student may have.
  5. Implement methods named setHomeworkScore and setQuizScore. Each of these methods takes 2 parameters. The first parameter specifies the index of the homework or quiz for which to set the score. The second parameter specifies the score on the homework or quiz. If an invalid index is passed to either of these methods it should print an error message. Note: The score for each homework or quiz will be an integer value indicating the percent of correct answers.
  6. Implement methods named getHomeworkScore and getQuizScore. Each of these methods takes 1 parameter indicating the index of the score to return. The score is returned as an integer. If an invalid index is passed to either of these methods it should print an error message and return the integer value -1.
  7. Create test methods for your GradeBook class. Your test methods must achieve statement coverage for your implementation.
