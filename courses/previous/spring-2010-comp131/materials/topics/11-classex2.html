Practice Exercises

  1. Implement a Song class that represents songs as follows:
    1. Use BlueJ to create a new project called MySong that is stored in your CS131 directory.
    2. Create a new class named Song in your MySong project.
    3. Define fields so that each song can keep track of the band name, the title of the song, the length of the song in minutes as a decimal (e.g. 3.45 minutes), and a rating for the song. The rating can be an integer number with higher numbers indicating a better song.
    4. Define a constructor for the Song class that has parameters which allow the band name, title and the length of the song to be initialized. New songs should have a rating of 0.
    5. Define accesor methods for each of the fields in the Song class.
    6. Define a method setRating that takes one parameter indicating a new rating for the song. This method should change the song's rating.

Homework Assignment 4a

  1. Implement a Movie class that represents movies as follows:
    1. Use BlueJ to create a new project called MyMovie that is stored in your CS131 directory.
    2. Create a new class named Movie in your MyMovie project.
    3. Define fields so that each movie can keep track of the title of the movie, the length of the movie in hours as a decimal (e.g. 2.15 hours), the MPAA rating of the movie (e.g. G, PG, R etc) and a star rating. The star rating will indicate how much you liked the movie. This star rating can be an integer number with higher numbers indicating more stars and a better movie (i.e. you liked a 5 star movie better than a 2 star movie).
    4. Define a constructor for the Movie class that has parameters that allow the movie title, the length of the movie and the movies' MPAA rating to be initialized. New movies should have a star rating of 0.
    5. Define accessor methods for each of the fields in the Movie class.
    6. Define a method setStarRating that takes one parameter indicating a new star rating for the movie. This method should change the movie's star rating.
    Turn in a printout of your Movie class definition as your solution for this question.
