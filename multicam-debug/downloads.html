Source code

A very preliminary version of the source code is available for download, as a Visual Studio 2010 project. It is intended that MultiCam will be released as an interactive open-source project sometime during 2012, but the code is not yet polished enough for programmers other than the original author to make meaningful contributions. The above version is provided primarily to reassure users that the project is indeed open source, and may also be of interest for those interested in browsing the code to understand how it works. However, is not yet recommended to edit the code or submit patches, with a major exception noted next.

The major exception is that Apple and Linux versions of the MultiCam app are definitely needed. As these can initially be developed as completely separate additions to the code base, anyone interested in developing an Apple or Linux MultiCam application is strongly encouraged to contact John MacCormick (jmac@dickinson.edu). Thanks!

Version notes

1.0.2.12:

  • Support for logging frame rates.
  • Support for (undocumented) multicam-devices.txt file for specifying which cameras to use.

1.0.2.6:

Minor bug fixes and efficiency improvements.

1.0.2.0:

  • Improved handling of cameras with non-standard resolutions.
  • Excluded several additional virtual camera drivers that are not presently compatible with MultiCam, including dvdriver, vhscrcap, softcam, virtualcam, and splitcam.

1.0.0.0:

Initial public release.
